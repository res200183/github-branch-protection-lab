# GitHub Branch Protection Lab

## Project Overview

This repository demonstrates GitHub Branch Protection and GitHub Actions best practices.

The goal of this project is to simulate a real-world software development workflow where code changes are protected by:

- Pull Requests
- Code Reviews
- Branch Protection Rules
- Automated CI checks
- GitHub Actions

## Technologies

- Git
- GitHub
- GitHub Actions
- Python
- CI/CD

## Branch Protection Features

This repository is configured to demonstrate:

- Require Pull Request before merge
- Require Approval before merge
- Require Status Checks to pass
- Require Conversation Resolution
- Restrict Direct Push to main

## CI Pipeline

GitHub Actions automatically:

- Checks out source code
- Installs dependencies
- Runs tests
